# Book Translator

Интерактивный TUI-инструмент для перевода ранобэ, веб-романов и других текстов с любого языка на русский. Поддерживает три LLM-бэкенда: **Gemini** (облачный), **Qwen** (облачный) и **Ollama** (локальный, бесплатно, без лимитов).

## Основные возможности

* **Три LLM-бэкенда**: переключение между облачными Gemini/Qwen и локальным Ollama через настройки в TUI.
* **Разные модели для разных задач**: каждый этап конвейера использует оптимальную модель.
* **Полноценный TUI-интерфейс**: текстовый интерфейс на базе Textual для управления всеми операциями.
* **Четырёхэтапный конвейер**:
  1. **Discovery** — поиск и извлечение терминов.
  2. **Translation** — перевод с учётом глоссария и контекста.
  3. **Proofreading** — вычитка каждого чанка.
  4. **Global proofreading** — финальная вычитка всей главы.
* **Управление глоссарием**: добавление, редактирование, импорт/экспорт терминов через TUI.
* **Пакетный перевод**: перевод всех ожидающих глав с отслеживанием прогресса.
* **Сохранение состояния**: SQLite-базы для возобновления прерванных переводов.
* **Кастомизация**: настройка промптов, стилей и параметров конвейера.
* **Stage-specific Temperature**: индивидуальная температура для каждого этапа конвейера.

## Настройка Ollama (локальный бэкенд)

> Пропустите этот раздел, если используете Gemini.

### 1. Установка

**macOS:**
```bash
# Через Homebrew
brew install ollama

# Или скачайте с https://ollama.com/download
```

**Windows:**
Скачайте установщик с https://ollama.com/download

### 2. Запуск

```bash
ollama serve
```

Проверка: откройте http://localhost:11434 в браузере — должна открыться страница "Ollama is running".

### 3. Загрузка моделей

```bash
ollama pull qwen3:8b          # ~5 GB — для поиска терминов
ollama pull qwen3:30b-a3b     # ~18 GB — для перевода и вычитки
ollama pull qwen3:14b         # ~9 GB — для глобальной вычитки
```

## Использование

### Запуск TUI

```bash
book-translator
```

Откроется главное меню (Dashboard). Если серия не инициализирована, нажмите `i` для создания новой.

### Главное меню (Dashboard)

| Клавиша | Действие |
|---------|----------|
| `Enter` | Перевести выбранную главу |
| `a` | Перевести все ожидающие главы |
| `i` | Инициализировать новую серию |
| `g` | Открыть глоссарий |
| `p` | Промпты |
| `c` | Настройки |
| `l` | Логи |
| `r` | Обновить |
| `q` | Выход |
| `Ctrl+D` | Переключить тему |

### Глоссарий

| Клавиша | Действие |
|---------|----------|
| `a` | Добавить термин |
| `Enter` | Редактировать выбранный |
| `d` | Удалить |
| `e` | Экспорт в TSV |
| `i` | Импорт из TSV |
| `f` | Поиск |
| `Esc` | Назад |

### Экраны приложения

- **Dashboard** — обзор серии, список глав, быстрый запуск перевода
- **Глоссарий** — управление терминами, импорт/экспорт
- **Прогресс перевода** — визуализация статуса глав по этапам конвейера
- **Пакетный перевод** — массовый перевод нескольких глав
- **Логи** — просмотр логов с фильтрацией по уровню и worker
- **Настройки** — конфигурация LLM, моделей, workers, rate limiting
- **Промпты** — просмотр и редактирование шаблонов промптов
- **Параметры перевода** — настройка формата вывода (DOCX/EPUB), чанкинга

## Конфигурация (`book-translator.toml`)

```toml
[series]
name = "My Series"
source_lang = "ja"
target_lang = "ru"

[llm]
backend = "ollama"  # или "gemini" или "qwen"
ollama_url = "http://localhost:11434"

[llm.models]
discovery = "qwen3:8b"
translation = "qwen3:30b-a3b"
proofreading = "qwen3:30b-a3b"
global_proofreading = "qwen3:14b"

[llm.options]
temperature = 0.3
num_ctx = 8192
think = false  # отключить режим мышления Qwen3

[llm.options.stage_temperature]
discovery = 0.1         # низкая для стабильного извлечения терминов
translation = 0.4       # выше для творческого перевода
proofreading = 0.3      # умеренная для вычитки
global_proofreading = 0.1  # низкая для консистивности

[qwen_cli]
model = "qwen-plus"

[gemini_cli]
model = "gemini-2.5-pro"

[workers]
max_concurrent = 3
max_rps = 100.0

[splitter]
target_chunk_size = 600
max_part_chars = 800
min_chunk_size = 300

[retry]
max_attempts = 3
wait_min_seconds = 4
wait_max_seconds = 10
```

## Стайлгайды

При инициализации автоматически копируется подходящий стайлгайд:
- `ja_ru.md` — Японский→Русский
- `ko_ru.md` — Корейский→Русский
- `zh_ru.md` — Китайский→Русский
- `en_ru.md` — Английский→Русский
- `default.md` — шаблон для других языков

---

## Кастомизация промптов

Создайте файлы в `{series_root}/prompts/`:
- `translation.txt` — промпт перевода
- `proofreading.txt` — промпт вычитки
- `global_proofreading.txt` — глобальная вычитка
- `term_discovery.txt` — поиск терминов

Пользовательские промпты имеют приоритет над встроенными.
